# Descarga Concurrente de Múltiples Archivos

## Descripción del Proyecto

Este proyecto implementa dos versiones de un descargador de archivos desde URLs:
- **Versión Secuencial**: Descarga los archivos uno tras otro
- **Versión Concurrente**: Utiliza múltiples hilos para descargar archivos simultáneamente

El objetivo es demostrar la mejora en el rendimiento al aprovechar la concurrencia en operaciones de I/O de red.

## Características

- Descarga múltiples archivos desde URLs públicas
- Comparación de tiempos entre ejecución secuencial y concurrente
